# Routix Frontend

Static frontend built with HTML/CSS/Vanilla JS.

## Scripts

- `npm run dev` - local static server on `http://localhost:5173`

## Configuration

Set backend URL in `frontend/js/runtime-config.js`:

```js
window.__ROUTIX_CONFIG__ = {
  API_URL: 'http://localhost:5000'
};
```

## Pages

- `/index.html`
- `/login.html`
- `/register.html`
- `/dashboard.html`
